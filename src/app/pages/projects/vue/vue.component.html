Razz Vue Rewards
About

I was given the challenge to create a Vue application as part of the application process for a company. Using my auth app as a base I was able to create a MEVN stack app in a few days with full auth and prize redemption limited to users on the server side.

Mevn stack app
  • Mevn Stack Front End with backend containing User Auth
  • Authentication via JWT using passportJs
  • Bcrypt Salted/Hashed password storage.
  • MongoDB database using mongoose ODM
  • Email service using NodeEmailer for verification/password reset emails
  • Prize redemption protected to logged in users only
  • Vue Front End
